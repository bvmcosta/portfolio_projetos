Sobre mim

Meu nome é Bruno Costa.

Sou biólogo com mestrado e doutorado na área da oceanografia e, atualmente, estudante de ciência de dados na Comunidade DS. Este portfólio inclui projetos de ciência de dados relacionados a temas ambientais (por exemplo, liberação de efluentes por estações de tratamento de esgoto) bem como a modelos de negócios de empresas de diferentes setores. O objetivo de cada projeto é adquirir experiência com as ferramentas da ciência de dados para auxiliar as empresas a diminuírem os impactos ambientais relacionados às suas atividades. Além disso, os projetos também objetivam propor soluções para problemas de negócio, resultando em redução de custo e aumento no lucro das empresas. Estou procurando oportunidade para trabalhar em empresas com modelos de negócio vinculados à temática ambiental.

Projetos em ciência de dados

Construção de um programa de fidelidade com clusterização de clientes.

Eu usei Python, Estatística e técnicas não-supervisionadas de Machine Learning para segmentar um grupo de clientes com base em suas características de performance de compra, a fim de selecionar grupos de clientes para formar um programa de Fidelidade com o objetivo de aumentar a receita da empresa.
